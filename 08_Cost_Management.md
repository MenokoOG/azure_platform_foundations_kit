# 08 — Cost Management (Don’t get surprised)

## Cost drivers
- compute hours (VMs, clusters)
- storage capacity + transactions
- network egress (outbound)
- premium SKUs

## Practical controls
- Tags: owner, env, project, cost_center
- Budgets + alerts
- Turn off dev compute overnight
- Delete unused resource groups
- Smallest SKUs for labs
