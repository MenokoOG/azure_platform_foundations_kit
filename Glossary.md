# Glossary

ARM: Azure Resource Manager
RBAC: Role Based Access Control
Entra ID: Azure identity system
VNet: Virtual Network
Subnet: Segment inside VNet
NSG: Network Security Group
SKU: Pricing tier/size
App Insights: App performance monitoring
Log Analytics: Central log store (KQL)
Private Endpoint: Private IP access to a PaaS service
ACR: Azure Container Registry
AKS: Azure Kubernetes Service
