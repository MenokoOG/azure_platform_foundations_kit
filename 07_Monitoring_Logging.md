# 07 — Monitoring & Logging (Observability)

## Key tools
- Azure Monitor: metrics/logs/alerts umbrella
- Log Analytics Workspace: central logs (KQL)
- Application Insights: app traces + dependencies
- Diagnostic settings: route logs/metrics

## Metrics vs Logs
- **Metrics**: numeric time-series (CPU, req/sec)
- **Logs**: events/records text/structured

## Minimum standard
- enable diagnostics on critical resources
- alerts for availability, error rates, latency, budgets
