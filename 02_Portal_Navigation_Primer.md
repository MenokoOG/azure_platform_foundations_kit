# 02 — Portal Navigation Primer (You should be fast here)

## Top 10 portal skills
1. Search bar (find any resource/service)
2. Switch subscription (context matters)
3. Resource group view (see what’s deployed)
4. “Create a resource” wizard
5. Keys & endpoints (where to find them)
6. Access Control (IAM) (permissions live here)
7. Activity log (who did what)
8. Diagnostic settings (send logs/metrics)
9. Cost Management + budgets
10. Deleting resources safely (start with RG)

## Where to look first when things break
- Resource → **Overview**
- Resource → **Activity log**
- Resource → **Metrics**
- Resource → **Diagnose and solve problems**
- Resource group → **Deployments** (ARM failures)
