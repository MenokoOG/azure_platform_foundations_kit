## How to use “graphics”
The `diagrams/*.md` files contain Mermaid diagrams.
Paste them into a Mermaid renderer (many Markdown apps support Mermaid) to view as visuals.

You can also render all diagram files to SVG locally:

```bash
npm install
npm run diagrams:render
```

Rendered files are written to `diagrams/rendered/`.
