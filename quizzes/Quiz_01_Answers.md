# Quiz 01 — Answers

1) MG → Subscription → RG → Resource
2) Subscription: billing/quota; RG: logical/lifecycle
3) Region: geography; Zone: separate datacenters in region
4) Azure Resource Manager control plane
5) Control: manage resources; Data: use the service
6) Roles assigned at scopes to grant access
7) Identity for a resource to access others without secrets
8) Private network
9) Metrics numeric series; logs events/records
10) Tags, budgets, smallest SKUs, shutdown schedules, delete RGs
